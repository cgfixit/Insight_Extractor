# `.codex/`

This directory holds Codex-specific onboarding material for Insight Extractor.

Purpose
- Keep repository-wide instructions in `AGENTS.md`.
- Keep lightweight operational templates and plugin setup notes in `.codex/`.

Key Files
- `codex_custom_instructions.md` - defaults for Codex behavior and review preferences.
- `ponytail-plugin.json` - plugin metadata used for optional review workflow setup.

Repository Setup Notes
- Read `AGENTS.md` before editing.
- Prefer smallest safe change for onboarding and docs-only tasks.
